# Personal Portfolio Website

Built with `Next.js`, `React`, `TypeScript`, and `Tailwind CSS`.

## Run locally

```bash
npm run dev
```

Then open `http://localhost:3000`.

## What to customize

- Replace `Your Name` and the hero copy in `src/app/page.tsx`
- Add your real email, GitHub, LinkedIn, and resume link
- Replace the sample project cards with your best work
- Update the skills and stack list so it matches your actual experience
